# LIB-1.0.0 Changelog

## Initial Release

**Released:** 2025-01-01

This is the initial release of the Xonaix Library.

### Included Units

- library/rust/mini-standard (1.0.0) - Rust language standard
- library/nats/mini-standard (1.0.0) - NATS messaging standard
- library/tauri/mini-standard (1.0.0) - Tauri desktop standard
- library/sql/mini-standard (1.0.0) - SQL database standard
- library/yaml/mini-standard (1.0.0) - YAML configuration standard
- library/toml/mini-standard (1.0.0) - TOML configuration standard
- library/wasm/mini-standard (1.0.0) - WebAssembly standard
- library/json/mini-standard (1.0.0) - JSON format standard
- library/c/mini-standard (1.0.0) - C language standard
- library/cpp/mini-standard (1.0.0) - C++ language standard
- library/meta/standards-index (1.0.0) - Standards index registry
- library/meta/xonaix-way (1.0.0) - Engineering philosophy standard
- library/meta/language-template (1.0.0) - Language standard template

### Notes

This is the initial batch sealing ceremony. All existing standards from xonaix-specs have been migrated to the Library structure as individual units.

Future releases will only include changed or new units per LIBRARY_SEALING_CONTRACT.md Section 6.2.
